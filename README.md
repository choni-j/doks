# 한손 투표

한 대의 휴대폰이나 태블릿을 여러 사람이 돌려가며 사용하는 현장 투표 앱입니다.

## 기능

- 설문 타이틀, 후보, 투표자 수 설정
- 후보 추가 및 삭제
- 투표자마다 후보 카드 순서를 무작위로 섞어서 표시
- 투표 직후 3초 가림 화면
- 모든 투표 완료 후 결과 공개
- 득표순 정렬, 공동 순위 처리, 1위 강조
- 결과 화면 PNG 저장 또는 모바일 공유
- 외부 서버나 외부 JavaScript 라이브러리 없이 동작
- 진행 데이터는 `sessionStorage`에만 저장되어 브라우저 탭/세션 종료 시 사라짐

## GitHub Pages 배포

1. 이 폴더의 파일을 GitHub 저장소 루트에 올립니다.
2. 저장소의 **Settings → Pages**로 이동합니다.
3. **Build and deployment**에서 `Deploy from a branch`를 선택합니다.
4. Branch를 `main`, 폴더를 `/(root)`로 선택하고 저장합니다.
5. 표시되는 GitHub Pages 주소로 접속합니다.

## 데이터 저장 방식

설문과 투표 진행 상태는 브라우저의 `sessionStorage`에 저장됩니다. 같은 탭을 새로고침하면 이어서 진행할 수 있지만, 탭이나 브라우저 세션을 종료하면 데이터가 사라집니다.
